## Overview

`gs26-smart-npc` is a lightweight JavaScript/HTML demo that showcases a **smart non‑player character (NPC)** using modern web technologies.  The project is built with Vite and runs a simple web‑based UI that can be extended or integrated into larger applications.

## Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/your‑org/gs26-smart-npc.git
   cd gs26-smart-npc
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run the development server**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser to see the demo.

4. **Build for production** (optional)
   ```bash
   npm run build
   ```
   The compiled files will be placed in the `dist/` folder.
